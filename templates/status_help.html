Plugin Status Images

API

The API will return an image response encoded as "image/png" as content-type, representing the status of the plugin requested in the URL.

/status/:name/

You must pass the following parameters to this query:

  • py: python version as used in tox, for instance "py33" or "py22".
  • pytest: pytest version, for instance "2.3.5" or "2.4.2".

For example, to query the status of plugin {{ name }} for python 3.3 and pytest version 2.4.2, hit this url:

/status/{{ name }}?py=py33&pytest=2.4.2
